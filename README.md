# lavolta
Space game project

## requirements
- Docker

## install

Развернуть приложение
```
bash entry.sh dev
```
Смотреть здесь (чтобы корректно работали сессии необходимо сделать алиас для localhost)
```
localhost:3020
```
Свернуть
```
bash entry.sh down
```
